# Expense Tracker — Трекер расходов

## 👤 Автор

## 📝 Описание программы
Expense Tracker — это графическое приложение для отслеживания личных финансов. Позволяет добавлять расходы, фильтровать по категориям и датам, а также подсчитывать общую сумму за выбранный период.

## 🛠️ Технологии
- Python 3.8+
- Tkinter (GUI)
- JSON (хранение данных)
- Git (контроль версий)

## 📦 Установка и запуск
### Через run.py (рекомендуемый способ):
```bash
python run.py
```
### Альтернативный способ:
```bash
python src/main.py
```
