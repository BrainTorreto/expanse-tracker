# Weather Diary - Дневник погоды

**Автор:** Фрицлер Денис Александрович

## Описание
GUI-приложение для ведения дневника погоды. Позволяет добавлять, просматривать, фильтровать записи о погоде с сохранением данных в JSON.

## Функционал
- ✅ Добавление записей (дата, температура, описание, осадки)
- ✅ Фильтрация по дате и температуре
- ✅ Сохранение/загрузка в JSON
- ✅ Валидация ввода
- ✅ Удаление записей
- ✅ Автосохранение

## Установка и запуск

### Требования
- Python 3.7+
- Tkinter (встроен в Python)

git clone https://github.com/BrainTorreto/expanse-tracker.git

# Запустить приложение
