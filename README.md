# Weather Diary - Дневник погоды

**Автор:** Фрицлер Денис Александрович

## Описание
GUI-приложение для ведения дневника погоды. Позволяет добавлять, просматривать, фильтровать записи о погоде с сохранением данных в JSON.

## Функционал
- Добавление записей (дата, температура, описание, осадки)
- Фильтрация по дате и температуре
- Сохранение/загрузка в JSON
- Валидация ввода
- Удаление записей

## Установка и запуск

### Требования
- Python 3.7+
- Tkinter (встроен в Python)

### Запуск
```bash
# Клонировать репозиторий
git clone https://github.com/BrainTorreto/expanse-tracker.git
cd expanse-tracker

# Запустить приложение
python run.py
```
### Примеры использования
Добавление записи:
Введите дату (2024-05-20)
Укажите температуру (15.5)
Напишите описание ("Солнечно, легкий ветер")
Отметьте осадки (если есть)
Нажмите "Добавить запись"

Фильтрация:
По дате: введите "2024-05-20" → покажет записи за этот день
По температуре: введите "10" → покажет записи с температурой выше 10°C

### Тесты
Проверка валидации:
Ввод неверной даты ("32.13.2024") → ошибка
Ввод текста в поле температуры → ошибка
Пустое описание → ошибка

Проверка фильтров:
Добавить записи с разными датами
Отфильтровать по дате → только нужные записи
Отфильтровать по температуре → корректные значения

Проверка сохранения:
Добавить записи
Закрыть и открыть приложение → данные сохранились
